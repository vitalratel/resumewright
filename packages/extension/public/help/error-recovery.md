# Error Recovery Guide

What to do when things go wrong and how to get back on track.

---

## Quick Recovery Steps

No matter what error you see, try these steps in order:

1. **Read the error message** - It tells you what's wrong and what to try
2. **Check suggestions** - Follow the "What you can try" list
3. **Try again** - Click "Try Again" if the error is temporary
4. **Refresh** - Reload the Claude.ai page
5. **Restart** - Close and reopen your browser
6. **Contact support** - If nothing works, get help

## Step-by-Step Recovery Scenarios

### Scenario 1: Syntax Error

**Error message:** "We couldn't read your CV code. There's a syntax error on line 42."

**What to do:**

1. **Look at the error details**
   - Note the line number (42 in this example)
   - Read the specific issue (e.g., "missing closing tag")

2. **Option A: Ask Claude to fix**
   ```
   "There's a syntax error on line 42 - missing closing tag. 
   Can you fix it?"
   ```
   
3. **Option B: Regenerate completely**
   ```
   "Can you create my CV again? Same content, please."
   ```

4. **Try converting again**
   - Click the ResumeWright icon
   - Conversion should work now!

---

### Scenario 2: Converter Won't Start

**Error message:** "The converter couldn't start. This usually happens if you're offline."

**What to do:**

1. **Check internet connection**
   - Are other websites loading?
   - Is Wi-Fi connected?

2. **Wait and retry**
   - Click "Try Again"
   - Extension might need a moment to load

3. **Refresh the page**
   - Reload the Claude.ai tab
   - Click the extension icon again

4. **Restart browser**
   - Close all browser windows
   - Open browser again
   - Go back to Claude.ai

5. **Check browser compatibility**
   - Chrome 90+ âœ“
   - Edge 90+ âœ“
   - Firefox (coming soon)
   - Safari âœ— (not supported)

### Scenario 3: CV Too Large

**Error message:** "Your CV is too large to process. Try reducing content."

**What to do:**

1. **Understand the limits**
   - Recommended: 1-2 pages (professional standard)
   - Maximum: ~10 pages
   - File size: <10 MB

2. **Option A: Reduce content**
   ```
   "My CV is too long. Can you help me reduce it to 2 pages? 
   Focus on most recent 10 years of experience."
   ```

3. **Option B: Split into multiple documents**
   - Main CV: 1-2 pages (essentials only)
   - Extended CV: Additional details
   - Publications list: Separate document
   - Portfolio: Separate document

4. **Option C: Simplify formatting**
   - Remove images
   - Use standard fonts
   - Reduce white space
   - Consolidate bullet points

---

### Scenario 4: Content Doesn't Fit

**Error message:** "We couldn't fit your content on the page."

**What to do:**

1. **Adjust page settings**
   - Click Settings (gear icon)
   - Try larger page size: Letter â†’ A4
   - Increase margins
   - Reduce font size

2. **Simplify content**
   ```
   "Some content is too wide for the page. Can you:
   - Break long paragraphs into shorter ones
   - Simplify the table
   - Reduce margin around sections"
   ```

3. **Check for very long lines**
   - Website URLs (shorten them)
   - Long job titles (abbreviate)
   - Wide tables (simplify)

4. **Try again**
   - After adjusting settings or content
   - Click "Convert to PDF"

---

### Scenario 5: Fonts Won't Load

**Error message:** "We couldn't load the fonts."

**What to do:**

1. **Check internet connection**
   - Fonts load from the web
   - Need active connection

2. **Use standard fonts**
   - Go to Settings
   - Choose Arial, Times New Roman, or Helvetica
   - Standard fonts always work

3. **Disable custom fonts**
   - Settings â†’ Custom Fonts
   - Turn off custom font loading
   - Stick with built-in fonts

4. **Try again**
   - Should work with standard fonts
   - Save settings first!

---

### Scenario 6: Unknown Error

**Error message:** "Something unexpected happened."

**What to do:**

1. **Try the basics**
   - Refresh page
   - Try again
   - Restart browser

2. **Test with different CV**
   - Ask Claude for a simple CV
   - Try converting that
   - If it works, issue is with specific CV content

3. **Check browser console** (for advanced users)
   - Press F12
   - Look for error messages
   - Include these when contacting support

4. **Contact support**
   - Provide error details
   - Screenshot if possible
   - Describe what you were doing
   - Mention browser version

## Emergency Backup

### If nothing works:

1. **Copy the CV content** (not the code)
   - Just the text from Claude's preview
   - Save in a text file

2. **Use alternative PDF creator**
   - Paste content into Google Docs
   - Format manually
   - Export as PDF

3. **Come back later**
   - Try ResumeWright again in a few hours
   - Issue might be temporary
   - Browser or extension might need update

4. **Report the issue**
   - Helps us improve
   - Others might have same problem
   - We can add fixes
